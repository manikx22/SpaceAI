# Portfolio Positioning

## One-line description

SpaceOps AI is an AI-driven satellite health intelligence platform for telemetry anomaly detection, failure forecasting, explainable fault analysis, self-healing recommendations, and digital twin action simulation.

## Resume-ready bullets

- Built a mission-control AI platform that predicts satellite subsystem faults from telemetry time-series data using PyTorch anomaly detection and LSTM-based failure forecasting.
- Designed a multistage decision layer with drift monitoring, fault severity staging, explainable AI, and rule-guided self-healing recommendations.
- Developed a digital twin simulation workflow to estimate post-action mission risk before execution.
- Delivered both a Streamlit operations dashboard and FastAPI inference service, with Docker packaging and automated tests.

## Demo emphasis

When presenting the project, emphasize:

- telemetry to decision pipeline
- explainability and trust
- digital twin simulation
- live context integration
- deployable engineering quality
